AI Node Tool Documentation

User Guide

The AI Node Tool was designed to create advanced AI in Unity using Nodes, Graphs and other built in functions.

  • Node Tool
  • AI Tool

Node Tool API

  • User Guide
  • Node Tool Reference

The Node Tool is a graph editor built on top of Unity's Graph View API.

Installation

To Install the Node Tool package simply Download the Unity Package on the Github and place it inside the assets folder.

This package does include a dependency which is the new Unity Input System.

Editor

To Create a new Graph Editor right click and under the Create tab, click Node Tool/ Base Graph Node. The Graph Node Editor is where you can create new Nodes by right clicking.

API

Classes

  • Base Node is the base node. Each node is split up into three different sub classes.
  • A Root Node is a node at the start of the tree.
  • A Pure Node is a node without a end pin.
  • A Single Node is a node which only allows one pin connection on the start pin and the end pin.
  • A Multi Node is a node which allows multiple pin connections on the start pin and the end pin.

There are script templates for each of these node types which can be found under the Create tab in the menu.

AI Tool

  • User Guide
  • AI Tool Reference

The AI Tool is a tool designed to create complex AI inside Unity.

Installation

To Install the AI Tool simply Download the Unity Package on the Github or grab the Git url and paste it into the package manager.

This package does include some dependencies which is the new Unity Input System.

Components

The AI Tool adds some new components which can be added to any object in the scene. If you selected a GameObject and click Add Component button there will be a new menu called AI node Tool. This menu will include all the current components currently avaible.
