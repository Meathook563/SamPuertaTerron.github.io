Actors and Scene

Fractured Engine currently only supports one scene which is created at the begining of the application. Each Scene contains a Entity Manager which holds the entities within that scene.

Fractured Game Engine uses a Entity Component System. Fractured Currently Supports 5000 Entities in a scene at once. An Entity in the Fractured is called a GameObject.

GameObjects can be created by calling the CreateEntity function, the actors will automatically get added to the scene with Transform Component. 

GameObject object;
object.id = Engine::GetInstance()->GetScene()->CreateEntity();

Actors can hold up to 32 components. To Add and Get a component you can use the following functions.

object.AddComponent<RenderComponent>();
object.GetComponent<RenderComponent>(); //Must have already added a component, otherwise this will return null.

Example

GameObject object;
object.id = Engine::GetInstance()->GetScene()->CreateEntity();
object.GetTransform()->Scale = { 10, 10 }; //Scales the object using a Vector2

List of Components

A list of components that can be used. A GameObject can hold up to 32 components, but not have multiple RenderComponents.

A TransformComponent used for moving actors around the scene. These are created by the engine within the CreateEntity() function. Each GameObject has a GetTransform() which returns the TransformComponent.

A RenderComponent used rendering objects to the screen.

Textures

A Texture is a sprite which can be applied to a GameObject. Textures which can be loaded into the Engine can be placed into the Resources/Sprites folder.

The GetTexture() function has an index parameter which is the position of the image inside the sprites folder. The number starts at 0 on the left, than increments right for each image. Fractured Contains a White Circle and a White Square to make it easier to propotype.

How to give a GameObject a Texture

object.AddComponent<RenderComponent>();
object.GetComponent<RenderComponent>()->Sprite = Engine::GetInstance()->GetRenderer()->GetTextures(0);//White Square 
//or
object.GetComponent<RenderComponent>()->Sprite = Engine::GetInstance()->GetRenderer()->GetTextures(1); //White Circle

Math

Vector 2

A Vector2D is the main vector within the game engine.

A Vector2 Constructs a new vector with given x, y components.

Vector2 newVector = Vector2(0.0f, 1.5f);
//or
Vector2 newVector = { 0.0f, 1.5f };

//Moves the Actor to 0.0f on the X and 1.5f on the Y
object.GetComponent<TransformComponent>()->Position.y = newVector;

Random Numbers

The Random class will generate random numbers.

RandomNumber which takes in a range parameter. This will generate a random number between 0 to the range.

int i = Random::RandomNumber(5);
float f = Random::RandomNumber(5.0f);

RandomNumberInRange which takes a low range and high range. This will generate a random number between those to values.

int i = Random::RandomNumberInRange(5, 100);
float f = Random::RandomNumber(5.0f, 100.0f);

Collisions

Fractured Only has a simple collision Function at the momenent. This is going to be replaced with a Box2D implementation.

The Functions checks if two GameObjects are colliding. 

bool bHasCollided = Collision::HasCollided(actor, actor2);
//or
if(Collision::HasCollided(actor, actor2))
{
    FR_LOG("Collided");
}

Input

Basic Keyboard Input

InputManager::IsKeyPressed(KeyCode::FK_W)

Key Codes included are the A - Z keys, Tab, Escape, Space, Left Shift and the arrow Keys.

Basic Mouse Input

InputManager::IsMouseButtonPressed(MouseCode::FM_LEFTMOUSEBUTTON)

Mouse Codes include only the Left Mouse Button and the Right Mouse Button

Home Page

When the engine is built an Empty Fractured-Sandbox project will also be created, this is the application project.

Inside here will be a folder named Resources. This is where Sprites can go. Also there will be 2 files named Sandbox.h and Sandbox.cpp. They will look like this.

Here is an example of a moving square.

Sandbox.h file


#pragma once
#include "Fractured.h"

using namespace Fractured;

namespace Sandbox
{
	class Sandbox : public Application
	{
	public:
		Sandbox()  = default;
		virtual ~Sandbox() = default;

	protected:
		/**
		 * \brief Return the WindowSettings used to Created the Window.
		 */
		WindowSettings GetWindowSettings() override;
		/**
		 * \brief Is Performed before at the start of the application.
		 */
		void OnAppCreate()				   override;
		/**
		 * \brief Runs every frame, where the main logic is handled.
		 * \param DeltaTime the time between frames.
		 */
		void OnAppUpdate(float DeltaTime)  override;
		/**
		 * \brief Runs every frame, where rendering Text can be handled.
		 */
		void OnAppRender()				   override;
		/**
		 * \brief Runs before the main engine closes. 
		 */
		void OnAppQuit()				   override;
	private:
		GameObject object;
		GameObject object2;
	};
}

std::shared_ptr<FApp> CreateApp()
{
	return std::make_shared<Sandbox::Sandbox>();
}

Sandbox.cpp

#include "Sandbox.h"

#include "Fractured/Actor/FActor.h"
#include "Fractured/Input/FInputManager.h"
#include "Fractured/Rendering/RenderManager.h"

namespace Sandbox
{
	WindowSettings Sandbox::GetWindowSettings()
	{
        //Where the user can Adjust the window settings. 
		WindowSettings windowSettings;
		windowSettings.title = "Fractured Application";
		windowSettings.width = 1280;
		windowSettings.height = 720;
		windowSettings.vsync = true;
		windowSettings.canResize = true;
		windowSettings.windowSettingsInternal.displayFps = true;
		return windowSettings;
	}

	void Sandbox::OnAppCreate()
	{
		//This is where the user would create GameObjects 
        object.id = Engine::GetInstance()->GetScene()->CreateEntity();
        object.AddComponent<RenderComponent>();
        object.GetComponent<RenderComponent>()->Sprite = Engine::GetInstance()->GetRenderer()->GetTexture(0); //This would load a white square which comes with the engine. 
		object2.id = Engine::GetInstance()->GetScene()->CreateEntity();
        object2.AddComponent<RenderComponent>();
		object2.GetTransform()->Position = { -5.0f, 8.5f }; //Sets the position to -5.0f on the x axis and 8.5f on the y axis
        object2.GetComponent<RenderComponent>()->Sprite = Engine::GetInstance()->GetRenderer()->GetTexture(1);
	}
	void Sandbox::OnAppUpdate(float DeltaTime)
	{
		//The Main Update Logic
        //Moving Objects

        if (InputManager::IsKeyPressed(KeyCode::FK_W))
		{
			object.GetTransform()->Position.y += 5.0f * DeltaTime; //Makes the object move up
		}
		if (InputManager::IsKeyPressed(KeyCode::FK_S))
		{
			object.GetTransform()->Position.y -= 5.0f * DeltaTime; //Makes the object move down
		}
		if (InputManager::IsKeyPressed(KeyCode::FK_A))
		{
			object.GetTransform()->Position.x -= 5.0f * DeltaTime; //Makes the object move left
		}
		if (InputManager::IsKeyPressed(KeyCode::FK_D))
		{
			object.GetTransform()->Position.x += 5.0f * DeltaTime; //Makes the object move right
		}

		if(Collision::HasCollided(object, object2))
		{
			//Sets a random positon when you collided with the object2
			object2.GetTransform().Position.x = Random::RandomNumberInRange(-8.0f, 8.0f); 
			object2.GetTransform().Position.y = Random::RandomNumberInRange(-8.0f, 8.0f);
		}
	}
	void Sandbox::OnAppRender()
	{
        //Where Text will be drawn
	}
	void Sandbox::OnAppQuit()
	{
        //Anything the user wants to do before the game quits
        //Runs before any of the engine quitting code
        //Logs (Only Displayed in Debug Configuration) that the Application is quitting and appends the window title to the end
		FR_LOG_INFO("Quitting Application: ", GetWindowSettings().title);
	}
}
